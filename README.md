# MedAssist

AI-powered health product recommender that analyzes symptoms and suggests OTC products and supplements.

## Live Site

[medassist-app-site.s3-website.us-east-2.amazonaws.com](http://medassist-app-site.s3-website.us-east-2.amazonaws.com)

## What it does

Describe your health symptoms and in return medassist will:
- Identify the likely condition
- Recommend products that are over the counter + supplements and describe its active ingredients
- Provide reasoning behind it's recommendations
- Inform when you should see a doctor
- Include a medical disclaimer

## Architecture

- **Frontend**: React, hosted on AWS S3
- **API**: AWS Lambda, API Gateway
- **AI**: Claude API (Anthropic) for health reasoning
- **Database**: DynamoDB for product data
- **Monitoring**: CloudWatch dashboards to display user metrics
- **Alerts**: SNS email notifications for any new input generated by user
- **Alarm**: CloudWatch alarm for API error detection

## Tech Stack

Python, JavaScript/React, AWS (Lambda, API Gateway, DynamoDB, S3, CloudWatch, SNS, IAM, EC2)
